# Kode Morse Sederhana

Program Python untuk melakukan enkripsi dan dekripsi kode Morse.

## Cara Menjalankan

Pastikan Python sudah terinstal di perangkat Anda.

Jalankan program dengan perintah:

```bash
python Turing.py
```

## Cara Penggunaan

### Enkripsi

1. Pilih menu enkripsi.
2. Masukkan teks yang ingin diubah menjadi kode Morse.
3. Program akan menampilkan hasil enkripsi.

Contoh:

```text
Masukkan teks: HELLO

Hasil:
.... . .-.. .-.. ---
```

### Dekripsi

1. Pilih menu dekripsi.
2. Masukkan kode Morse yang ingin diterjemahkan.
3. Pisahkan setiap huruf dengan spasi.
4. Program akan menampilkan hasil dekripsi.

Contoh:

```text
Masukkan kode Morse:
.... . .-.. .-.. ---

Hasil:
HELLO
```

## Contoh Kode Morse

| Karakter | Morse |
|-----------|--------|
| A | .- |
| B | -... |
| C | -.-. |
| 1 | .---- |
| 2 | ..--- |
| 3 | ...-- |
